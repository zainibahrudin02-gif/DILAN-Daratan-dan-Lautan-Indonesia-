# DILAN-Daratan-dan-Lautan-Indonesia-
DILAN (Daratan dan Lautan Indonesia) merupakan media pembelajaran interaktif berbasis digital untuk membantu siswa kelas V mempelajari materi IPAS. Aplikasi ini memuat topik letak geografis dan astronomis, batas wilayah, negara kepulauan, pulau-pulau di Indonesia, serta kondisi geologis. Dilengkapi menu belajar, kuis, dan evaluasi yang menarik.
